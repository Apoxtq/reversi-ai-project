## 💡 经验教训

### Week 0-1见解
1. **构建系统：**
   - CMake功能强大但有平台特定问题
   - 备用构建方法 (batch脚本) 很有价值

2. **C++标准：**
   - C++20特性显著改进bitboard代码
   - 需确保编译器支持 (GCC 13.2+良好)

3. **项目组织：**
   - 清晰的目录结构有助于开发
   - 关注点分离 (core/ai/ui) 很重要

4. **文档：**
   - 边做边记录，不要事后补
   - 学习笔记帮助巩固理解
